# Public Repo

## Java codestyle

To use this codestyle in vscode you will need to install the "Language Support for Java(TM) by Red Hat" extension.

Then add the following to your settings.json (Ctrl+Shift+P > Preferences: Open Settings (JSON)):

```json
  "java.format.enabled": true,
  "java.format.settings.url": "https://raw.githubusercontent.com/ecmarkets-uk/public/a38c9539086416a231ebe6461afab55c1c1fc106/codeStyle.xml",
```
